# إعداد موقع رحيق الحجامة

## الملفات المطلوبة

ضع هذه الصور في نفس المجلد:

| اسم الملف | الوصف |
|-----------|-------|
| `logo.png` | شعار رحيق الحجامة (الصورة التي أرسلتها) |
| `gallery1.jpg` | صورة المركز الأولى (الحجامة النارية) |
| `gallery2.jpg` | صورة المركز الثانية (أكواب الحجامة) |

## نشر الموقع

### خيار 1 — Netlify (مجاني ومباشر)
1. اذهب إلى https://netlify.com
2. اسحب المجلد كاملاً وأفلته في الصفحة
3. ستحصل على رابط فوراً

### خيار 2 — GitHub Pages (مجاني)
1. أنشئ مستودعاً على GitHub
2. ارفع الملفات
3. فعّل GitHub Pages من الإعدادات

## إضافة رقم الهاتف
في ملف `index.html` ابحث عن:
```
"+212"
```
واستبدلها برقم هاتفك الحقيقي.
